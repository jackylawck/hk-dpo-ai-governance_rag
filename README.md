### 🧱 Governance Methodology & Architecture (管治方法論與架構)

This project is powered by the **"Jacky Law Human-Centred AI Governance Stack"** and features Defense-in-Depth engineering controls to protect information supply chains:
本專案採用 **「Jacky Law 人本 AI 管治五層架構」** 進行頂層設計，並實作了多層防禦工程控制，以捍衛企業資訊供應鏈安全：

1. **ISO 42001 Cryptographic Audit Trail (密碼學審計軌跡):** Every query and retrieved text chunk automatically generates a SHA-256 cryptographic hash ID, recorded in a non-repudiable audit ledger for immutable verification.
2. **Data Minimization & In-Memory Sandbox (資料最小化與內存沙盒):** Documents are indexed 100% locally in the browser session's RAM using open-source `sentence-transformers` and `FAISS`. No files are written to cloud storage or transmitted to third-party proprietary APIs, completely eliminating data leakage (PII Risk).
3. **Failure-Safe Rule Alignment (防幻覺確定性控制):** The retrieval engine bypasses the processing layers of large language models, enforcing a strict "what you upload is what you query" deterministic defense. It filters out irrelevant noise and prevents compliance illusions.
